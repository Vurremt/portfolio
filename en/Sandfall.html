The original texture

As part of the #MadeWithSubstance initiative, which highlights original creations by artists using Adobe Substance 3D software, and with the release of the game Clair Obscur: Expedition 33 coinciding with my internship, I was able to see all the 3D artistic productions from the game appear on ArtStation over time.

There were beautiful creations made with Substance Painter, but personally, I was more drawn to the ones created in the software I was working on: Substance Designer. The creations that particularly stood out to me were those by Alexandre Breton, where I was amazed by what he could achieve with the software.
>> This modular texture in an external tool impressed me

Another creation that impressed me was an ink sea texture animated via shaders in Unreal Engine, since it's not possible to do that in Designer. This struck me especially because it was the subject of my internship.
>> The animated texture

My attempt to recreate it

So I tried to recreate it, with my own vision, using the animation system I developed during my internship.

Starting from a basic water texture, I tried to add waves, work on reflectivity to get closer to an ink texture, and added variations with stagnant dark oil spots, typical of ink that has been spread for too long.

I also tried to work on height and normals to get realistic shapeless blobs, but I struggled to reproduce the behavior of the original texture — mine still looks too much like "waves" and not enough like "blobs".

Finally, the animation was added using the system I developed, unfortunately confidential, but which allows for this subtle movement effect. I tried, like in the original texture, to give a sense of motion without displacement — meaning the ink blobs always return to their original positions to avoid inconsistencies.
